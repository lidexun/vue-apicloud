# 使用的框架

- 基于 Vue-cli5
- Vue3 + Vue-router6 + Vant UI
- Vue3 自动导入 composition api
- Vue-router 根据文件系统生成路由地址
- 移动端适配方案(REM)
- Less 预处理
- Axios 封装与配置
